Cisco Packet Tracer y Ayudantía T3

Ariel Martínez Silberstein

Estimad@s,

Mañana publicaremos la T3. Para esta tarea usaremos Cisco Packet Tracer, una aplicación bastante versátil para diseñar y simular redes de computadores. La ayudantía correspondiente a esta tarea será este viernes 26 a las 8:30. Les recomendamos que, para sacarle el máximo provecho, ya tengan instalado el software en sus computadores para entonces.

Además, aprovechamos de recordarles que el deadline del P2 es hoy a las 23:59 a través de la master branch de sus repositorios en GitHub (y además a través del servidor del curso para quienes hayan desarrollado el bonus 3).

¡Mucho ánimo!

Raimundo y Ariel
